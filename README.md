## üî• **What is this?**

**Git Intel** isn't just another GitHub stats viewer. It's a **comprehensive telemetry dashboard** for your coding life. 

We don't just count your commits; we analyze your **soul** (or at least your code quality). Using advanced AI (Groq/Gemini), Git Intel adopts the persona of a *hard-to-impress Ex-Google Engineering Manager* to give you:

*   **SWOT Analysis**: Strengths, Weaknesses, Opportunities, and Threats based on your actual code.
*   **Hiring Odds**: A harsh calculation of your probability of passing a FAANG screen.
*   **Vital Signs**: Real-time telemetry on your commit velocity, PR ratio, and community impact.
*   **Ghost Code**: Detection of abandoned projects and "tutorial hell" repositories.

## üöÄ **Getting Started**

### Prerequisites
*   Node.js (v18+)
*   A GitHub Token (optional, for higher rate limits)
*   Groq/Gemini API Keys (for the roasting)

### Installation

1.  **Clone the repo**
    ```bash
    git clone https://github.com/PranavShirke/GitInsight.git
    cd GitInsight
    ```

2.  **Install Dependencies**
    ```bash
    # Backend
    cd backend
    npm install

    # Frontend
    cd ../frontend
    npm install
    ```

3.  **Configure Environment**
    Create a `.env` file in `backend/`:
    ```env
    PORT=5000
    GITHUB_TOKEN=your_github_token
    GROQ_API_KEY=your_groq_key
    GEMINI_API_KEY=your_gemini_key
    ```

4.  **Run It**
    ```bash
    # Terminal 1 (Backend)
    cd backend
    node server.js

    # Terminal 2 (Frontend)
    cd frontend
    npm run dev
    ```
